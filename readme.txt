# The Grill - A MAC Attack List Builder

An unofficial, browser-based roster builder and interactive digital tracker for MAC Attack, the sci-fi miniature wargame by Chris McDowall.

This tool is designed to make list-building fast and to provide a clean sheet for tracking your units during a game, online or on paper.

## Features

* Build your force with built-in rule validation.
* Clickable checkmarks to track damage and ammo directly on your device during play.
* Export your rosters and individual units to JSON files and load them later. 
* Upload custom photos of your miniatures and assign custom names to units and individual weapons.
* Assign playing cards directly to units for initiative tracking.

## Offline Use
Because the entire application is contained within a single file, you can simply download `index.html` and open it in any modern web browser.

## Technical Details

This project is built as a single-file web application using:
* HTML5
* React (via CDN)
* Tailwind CSS (via CDN)

It uses your browser's local storage and object URLs to handle photo uploads and data saving, meaning your custom lists and photos are never uploaded to an external server.
